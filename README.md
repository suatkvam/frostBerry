# FrostBerry

A 2D platformer game created for **SNOWBERRYGAMEJAM** (December 26-28, 2025)

## About

FrostBerry is a platformer game built around the theme of **"Rewind"**. Navigate through challenging levels using your rewind ability to overcome obstacles and defeat enemies.

## Controls

- **Arrow Left/Right**: Move left/right
- **Arrow Up**: Jump
- **Space**: Attack
- **Shift**: Rewind

## Game Jam

- **Event**: SNOWBERRYGAMEJAM
- **Duration**: 3 days (December 26-28, 2025)
